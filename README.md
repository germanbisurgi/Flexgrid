# flexgrid

##syntax sugar

Bootstrap:
```html
col-xs-12 col-xs-offset-2
```

Flexgrid:
```html
xs-12 xs-off-2
```

## no need a container


Bootstrap:


```html
<div class="container-fluid">
    <div class="row">
      <div class="col-sm-4"></div>
      <div class="col-sm-4"></div>
      <div class="col-sm-4"></div>
    </div>
</div>
```
FlexGrid:


```html
<div class="row">
    <div class="sm-4"></div>
    <div class="sm-4"></div>
    <div class="sm-4"></div>
</div>
```
