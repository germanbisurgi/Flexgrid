FlexGrid

A very flexible, fluid and responsive grid system based on flexbox

Rows and columns

Use the flex-row class to create a row.

Now tha you have a row you can add Columns to it. Use the flex-xs-4 class to create a column that will be 4/12 of the that row.

Every Flexgrid class have a flex- prefix.


<div class="flex-row">
    <div class="flex-xs-6">
        <!-- content -->
    </div>
</div>

Responsive Modifiers

Flexgrid uses responsive modifiers: xs (extra small), sm (small), md (medium) and lg (large) into predefined classes to control the columns behaviors in relation with the width of the viewport.


<div class="flex-row">
    <div class="flex-xs-12 flex-sm-3  flex-md-4 flex-lg-4"></div>
    <div class="flex-xs-6  flex-sm-6  flex-md-4 flex-lg-8"></div>
    <div class="flex-xs-6  flex-sm-3  flex-md-4 flex-lg-8"></div>
    <div class="flex-xs-4  flex-sm-6  flex-md-4 flex-lg-4"></div>
    <div class="flex-xs-4  flex-sm-6  flex-md-4 flex-lg-3"></div>
    <div class="flex-xs-4  flex-sm-12 flex-md-4 flex-lg-9"></div>
</div>

Offsets

Offset a column. Ex. flex-xs-offset-4


<div class="flex-row">
    <div class="flex-xs-11 flex-xs-offset-1""></div>
    <div class="flex-xs-10 flex-xs-offset-2"></div>
    <div class="flex-xs-9 flex-xs-offset-3"></div>
    <!-- etc -->
</div>

Nesting

Yes, we can


<div class="flex-row">
    <div class="flex-xs-6">
        <div class="fle-row">
            <div class="flex-xs-6">
                <!-- etc -->
            </div>
        </div>
    </div>
</div>

Justify

This defines the alignment along the x axis



<div class="flex-row flex-xs-justify-start">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-justify-center">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-justify-end">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-justify-between">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-justify-around">
    <div class="flex-xs-1"></div>
</div>

Align

This defines the default behaviour for how flex items are laid out along the cross axis on the current line.



<div class="flex-row flex-xs-align-start">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-align-center">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-align-end">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-align-baseline">
    <div class="flex-xs-1"></div>
</div>



<div class="flex-row flex-xs-align-stretch">
    <div class="flex-xs-1"></div>
</div>

Self

This allows the default alignment to be overridden for individual flex items.



<div class="flex-row">
    <div class="flex-xs-1 flex-xs-self-start"></div>
    <div class="flex-xs-1 flex-xs-self-center"></div>
    <div class="flex-xs-1 flex-xs-self-end"></div>
    <div class="flex-xs-1 flex-xs-self-baseline"></div>
    <div class="flex-xs-1 flex-xs-self-stretch"></div>
</div>
